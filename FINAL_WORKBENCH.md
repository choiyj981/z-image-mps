## 🚀 [NEW] 블로그 실전 능력 검증 (Capability Test)

블로그 포스팅에 필수적인 **4대 핵심 카테고리**에 대해 실제 사용 가능한 품질인지 검증했습니다.

| 카테고리 | 주제 | 테스트 결과 | 추천 프롬프트 키워드 |
|----------|------|-------------|----------------------|
| **맛집 (Food)** | 삼겹살, 라떼 | ✅ **매우 우수** | `steam rising`, `delicious texture`, `warm lighting` |
| **여행 (Travel)** | 제주도, 야경 | ✅ **우수** | `wide angle shot`, `cinematic lighting`, `landmark` |
| **제품 (Product)** | 스마트폰, 화장품 | ✅ **매우 우수** | `clean background`, `studio lighting`, `high detail` |
| **정보 (Info)** | 코딩, 금융 | ✅ **우수** | `3D render`, `isometric`, `concept art` |

> **결론**: 별도의 복잡한 설정 없이 **기본 설정(Steps 5)**만으로도 블로그 상위 노출에 손색없는 이미지가 생성됩니다.

## 🛠️ 블로그 자동화 연동 가이드

이제 외부 API(Gemini, Pollinations) 대신 **로컬 Z-Image**를 사용하여 비용 0원으로 무제한 이미지를 생성하세요.

### 1. 서버 실행 (터미널 1)
```bash
uv run z-image-server
```
> **팁**: 이 창을 켜두면 모델 로딩 시간(60초)이 사라져서, 요청 즉시 1초 만에 생성이 시작됩니다.

### 2. 블로그 프로그램 연동 (Python)
```python
import requests

def generate_blog_image(prompt):
    response = requests.post("http://localhost:8000/generate", json={
        "prompt": prompt,
        "width": 896,
        "height": 512,
        "steps": 5
    })
    return response.json()["image_path"]
```
