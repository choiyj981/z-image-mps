## 🎯 최종 추천 가이드

### 1️⃣ "가장 빠르게 많이 만들고 싶다" (속도형)
- **설정**: **768 x 432, Steps 5**
- **성능**: 1장당 55초 / 100장 1시간 30분
- **특징**: 블로그 모바일 뷰에 최적화된 속도

### 2️⃣ "품질과 속도 모두 챙기고 싶다" (밸런스형) - ⭐ 강력 추천
- **설정**: **896 x 512, Steps 5**
- **성능**: 1장당 1분 40초 / 100장 2시간 47분
- **특징**: PC/모바일 모두 보기 좋은 16:9 비율

### 3️⃣ "크기가 깡패다! 제일 크게!" (고해상도형)
- **설정**: **1280 x 720 (HD), Steps 5**
- **성능**: 1장당 2분 44초 / 100장 4시간 30분
- **특징**: 꽉 찬 HD 화질. 시간은 좀 걸리지만 임팩트 있음.

## 🚀 바로 실행하기

터미널에서 아래 명령어를 입력하여 인터랙티브 모드를 실행하세요:

```bash
uv run z-image-interactive
```

**추천 설정 입력:**
```bash
# 밸런스형 (추천)
set steps 5
set size 896 512

# 고해상도형 (HD)
set steps 5
set size 1280 720
```

이제 프롬프트만 입력하면 이미지가 계속 생성됩니다!
